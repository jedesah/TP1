Introduction

Il s'est produit, il n'y a pas si longtemps, une catastrophe de nature apocalyptique qui a replonge la societe humaine a l'age noir du savoir. Il reste encore certains artefact de l'ancienne civilization, mais trop peu de gens savent comme faire fonctionner ces machins de technologie.

C'est ici que tu fais ton entre, tu es le magicien informatique de ta tribue de nomade. Le chef Bram de la tribu a besoin d'une application de "to do" pour faire la gestion des taches de reconstruction de l'humanite. Il fait appel a tes pouvois pour lui creer une telle application.

Tu es un peu inquiet de ne pas reussir a effectuer la tache qui est demande de toi, mais pas de souci, je serai la pour t'aider dans ton apprentissage de la sorcellerie informatique.

Unix

Tous les ordinateurs Windows ont ete detruits par l'apocalypse. De toute facon, Windows 9 etait un flop total.

En tant que magicien de l'informatique, tu te veras confronte a la console console Unix. Il serait bien de savoir un peu comment ca fonctionne. Pour l'instant, tu peux te contenter de savoir que ls te permet de lister les fichiers dans le repertoire courant et cd te permet de changer de repertoire courant (change directory). Afin de completer notre tache, nous allons utiliser deux logiciels particuliers, Git et Play. pour savoir si le logiciel est present sur la machine et pour savoir c'est quel version. Pour plus d'information sur Git, tu peux visiter le site web officiel.

Nous avons choisi de faire notre application de "todo" pour le secours de l'humanite sous la form d'une application web afin que le chef Bram puisse verifier c'est todo meme lorsqu'il est partie en conquete de territoire lointain, sur n'importe quel terminal connecte au web magique. Allez de l'avant et extrayez l'archive et placez le dossier a quelque part dans votre repertoire personnel.

De simple debut

Choisissez un emplacement adequat comme ~/Projet et executer la commande:

$ play new todo

Play vous demandera de confirmer le nom de l'application, vous pouvez simplement cliquer sur Entrez pour garder le nom todo. Ensuite choisissez l'option 1(Create a simple Scala application). Play devrait avoir cree un repertoire denomme todo dans le repertoire courant. Rentrez dans ce repertoire et executez la commande:

$ play

Ceci a commence la console Play. Executez maintenant la commande:

$ [todo] $ run

afin de partir l'application web en debug. Vous pouvez maintenant visiter l'adresse localhost:9000 dans votre navigateur afin de voir a quoi ressemble le site web qui sert d'exemple
